Comment jouer au démineur

Le but du jeu est de marquer toutes les mines.
Cliquez sur un carré pour le révéler.


Pour marquer une mine, effectuez un clique droit sur un carré.
Un espace vide signifie qu'il n'y a pas de mines adjacentes.


Le jeu se termine lorsque toutes les mines sont marquées ou que vous révélez une mine. Utilisez les chiffres pour savoir où sont les mines et marquez-les.


Le chiffre qui s'affiche sur les cases cliquées indique le nombre de mines se trouvant à proximité : à gauche ou à droite, en haut ou en bas, ou en diagonale.

A vous de jouer :

Vous avez toutes les clés en main pour réussir au jeu du démineur !
